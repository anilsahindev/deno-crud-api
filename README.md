# TODOS

| Route | HTTP Verb	 | POST body	 | Description	 |
| --- | --- | --- | --- |
| /todos | `GET` | Empty | List all todos. |
| /todos | `POST` | {'text':'working deno' } | Create a new todo. |
| /todos/:todoId | `PUT` | Empty | Update a todo. |
| /todos/:todoId | `DELETE` | Empty | Delete a todo. |
